Python Commands for Session 1 exercise

# Open Python mode from command terminal
python

# exit from python mode
exit()

# check the version of python
python --version

# list the installed module in this package
pip list

# install a dependency module
pip install "module_Name"

# install dependency modules from requirements.txt
pip install -r requirements.txt

Window Commands 

# list window commands
help

# list files in the directory 
dir

# Change to sub directory
cd "sub_directory_name"

# make a sub_directory in the present directory
mkdir
